# Colombo private workspace area

This directory is intentionally mostly empty in the public template.

In a private Colombo clone, onboarding and owner-approved improvement sessions may add:

- `connected-systems/*.md` — system cards for real systems exposed through MCP.
- `test-messages/*.md` — ready-to-copy Slack prompts generated for each connected system.
- `runbooks/*.md` — longer runbooks when `AGENTS.md` would become too large.
- `improvements/pending/*.md` — nightly review suggestions waiting for owner approval.
- `improvements/applied/*.md` — trace records for approved changes.

Do not publish private connected-system cards, customer data, feedback history, credentials, or internal runbooks in the open-source template.
